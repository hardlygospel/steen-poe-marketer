## Quick start

```bash
# macOS / Linux
chmod +x setup.sh && ./setup.sh

# Or directly with Python 3.9+
python3 steen_poe.py

# Jump straight to a league
python3 steen_poe.py --league "Settlers"

# Search immediately without the menu
python3 steen_poe.py --search "Headhunter"
```

All three dependencies install automatically on first run.

| Key | Action |

## The 10 modules

### 1 вҖ” Currency Analysis вҡ–
Live buy/sell spreads for every currency on poe.ninja. Shows chaos equivalent, discount vs median, and в–І/в–ј/в”Җ 7-day trend. Items marked **вҳ…** are trading below the median вҖ” a potential buy opportunity.

### 2 вҖ” Arbitrage Finder рҹ”„
Finds profitable round-number trades between currency pairs. Surfaces pairs where the nearest whole-number exchange ratio gives a better deal than the exact ninja rate. Items marked **рҹ”Ҙ** have >10% margin.

### 3 вҖ” Divination Cards рҹғҸ
All divination cards with set cost (in chaos), reward description, stack size, and 7-day trend. Filter by minimum set cost. Cards worth вүҘ100c are highlighted in gold.

### 4 вҖ” Gem Flipper рҹ’Һ
Profit margin between a base gem and its max level (20) / quality (20%) version. Filter by minimum profit. Shows ROI %, highlighted green (>100% ROI) or bright green (massive flip).

### 5 вҖ” Economy Overview рҹ“Ҡ
Top-10 snapshot across 8 categories: Currency, Divination Cards, Unique Weapons, Unique Armours, Unique Accessories, Skill Gems, Unique Flasks, Unique Jewels.

### 6 вҖ” Stash Pricer рҹҸҰ
Enter item names one at a time. Fuzzy-matches against the full poe.ninja database. Shows price per item and a grand total.

### 7 вҖ” Bulk Calculator рҹ§®
Enter stacks using shorthand aliases (e.g. `40 div`, `200 chaos`) to get the total value in both chaos and divine. Accepts one entry per line; empty line or `done` to calculate.

### 8 вҖ” Watchlist рҹ‘Ғ
Track any items across sessions. Each time you open the watchlist it shows the current price alongside the last-seen price and the delta вҖ” green for gains, red for losses.

### 9 вҖ” Top Movers рҹ“Ҳ
Scans every category for the biggest % gainers and losers over the last 7 days. Good for spotting early meta shifts before the price peaks.

### 10 вҖ” Item Search рҹ”Қ
Fuzzy search by partial name across all categories simultaneously. Ranked by price. Also accessible directly via `--search` on the command line.

## Reading the data

| Symbol | Meaning |
|---|---|
| `в–І +X.X%` | Price rising over 7 days |
| `в–ј вҲ’X.X%` | Price falling over 7 days |
| `в”Җ 0.0%` | Stable price |
| `вҳ…` | Buying below median вҖ” opportunity |
| `рҹ”Ҙ` | Arbitrage margin >10% |
| `c` | Chaos Orb |
| `div` | Divine Orb |
| `k` / `M` | Thousands / millions |

**Column guide:**

| Column | Module | Meaning |
|---|---|---|
| Spread | Currency | Buy/sell gap вҖ” lower = more liquid |
| Discount | Currency | vs median вҖ” negative = bargain |
| Margin | Arbitrage | Profit % from round-number trade |
| Set Cost | Div Cards | Full card set value in chaos |
| ROI | Gem Flipper | Profit as % of base cost |
| Delta | Watchlist | Price change since last check |

## POESESSID authentication

POESESSID is **completely optional**. All 10 modules work without it вҖ” poe.ninja is a public API.

POESESSID is only used for `api.pathofexile.com` requests (league enumeration). Without it, leagues come from a poe.ninja fallback list.

**To find your POESESSID:**
1. Log in at [pathofexile.com](https://www.pathofexile.com)
2. Open DevTools вҶ’ `F12` вҶ’ Application вҶ’ Cookies
3. Copy the value next to `POESESSID`

**To add it:** run the app вҶ’ press `S` for Settings вҶ’ option `1`.

Stored in `~/.steen_poe/config.json`. Never sent to poe.ninja вҖ” only to GGG's official API.

## Configuration and cache

| Path | Contents |
|---|---|
| `~/.steen_poe/config.json` | League, POESESSID, watchlist, last-seen prices |
| `~/.steen_poe/cache/` | poe.ninja responses (15-min TTL, one file per endpoint) |

The Market Pulse panel in the main menu shows cache age so you always know how fresh the data is. Clear the cache via `S вҶ’ Settings вҶ’ 2`.
